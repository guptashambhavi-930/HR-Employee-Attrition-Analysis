# HR Employee Attrition Analysis

## Project Overview

This project analyzes employee attrition using SQL, Python, Power BI, and Machine Learning.

The objective is to identify the major factors responsible for employee attrition and build a predictive model to classify whether an employee is likely to leave the company.
